# swift m3u8 tools

A small collection of tools for working with m3u8 files (as defined by https://www.rfc-editor.org/rfc/rfc8216.html) in swift. Currently just a parser and printer which should make transforming m3u8 files easier.

Includes extensions for image-only playlists defined by https://github.com/image-media-playlist/spec/blob/master/image_media_playlist_v0_4.pdf
